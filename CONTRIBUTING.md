## Come Contribuire

### Segnalare Bug

Se trovi un bug, per favore apri una [issue](https://github.com/TUO_USERNAME/gardenos/issues) con:
- Descrizione chiara del problema
- Passi per riprodurre il bug
- Comportamento atteso vs comportamento reale
- Screenshot se applicabile
- Informazioni sull'ambiente (browser, OS, versione)

### Proporre Feature

Le proposte di nuove feature sono benvenute! Apri una [issue](https://github.com/TUO_USERNAME/gardenos/issues) con:
- Descrizione dettagliata della feature
- Caso d'uso e motivazione
- Esempi di come dovrebbe funzionare

### Pull Requests

1. Fai un fork del repository
2. Crea un branch per la tua feature (`git checkout -b feature/amazing-feature`)
3. Committa le tue modifiche (`git commit -m 'Add amazing feature'`)
4. Pusha sul branch (`git push origin feature/amazing-feature`)
5. Apri una Pull Request

### Linee Guida per il Codice

- Segui lo stile di codice esistente
- Aggiungi commenti dove necessario
- Assicurati che il codice funzioni su mobile e desktop
- Testa le tue modifiche prima di fare il commit

### Setup per lo Sviluppo

Vedi [README.md](README.md) per le istruzioni di installazione e setup.

## Domande?

Se hai domande, apri una [discussion](https://github.com/TUO_USERNAME/gardenos/discussions) o una [issue](https://github.com/TUO_USERNAME/gardenos/issues).
